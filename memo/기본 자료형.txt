형 변환

자동형 변환(묵시적형변환)
1. 작은 자료형 -> 큰 자료형
2. 문자형 데이터 -> 정수

강제형 변환(명시적형변환)
2. 큰 자료형 -> 작은 자료형
3. 정수 -> 문자
* 실수를 정수로 바꿀 때 소수점 아래의 값은 소실된다.

	컴파일
자바코드 -> 바이트 코드 -> JVM이 실행 -> 결과가 출력

자바는 실행 성능을 향상시키기 위해 컴파일 단계에서 연산을 수행

데이터의 입력
java.util에 있는 Scanner 클래스를 사용해야 된다.

import
- 다른 패키지에 있는 클래스를 사용하겠다고 명시하는 것

import java.util.Scanner;

클래스 안에는 함수(기능)가 정의되어 있다.

클래스 안에 정의된 메서드를 사용하기 위해서는 객체를 생성해야 한다.

객체를 생성하는 방법(new 클래스명();)
int x = 100;
char c = 'a';
Scanner sc = new Scanner(System.in);

System.in ->  키보드 입력을 받는 기본 통로

만들어진 객체를 통해서 클래스 안에 정의되어 있는 메서드를 호출할 수 있다.

sc.메서드명();
