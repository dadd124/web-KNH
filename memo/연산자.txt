연산자
연산기능이 있는 기호
산술연산자 : +, -, *, /(몫), %(나머지)
비교연산자 : ==, !=, <, <=, >, >=
논리연산자 : &&, ||, !
증감연산자 : ++, --
비트연산자 : &, |, ^, ~
시프트연산자 : <<, >>, <<<, >>>

조건문
- 조건에 따라 실행할 명령을 결정한다.
분기를 정하는 것
- if문
- if -> 조건에 따라 실행할지 말지 결정
- if - else -> 참일 때 실행할 명령, 거짓일 때 실행할 명령
- if - else if -> 여러 조건을 비교하여 실행할 명령을 결정

반복문
- 조건에 따라 명령을 반복할지 말지 결정
